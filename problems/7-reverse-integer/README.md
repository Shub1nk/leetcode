# 7. Reverse Integer

**Сложность:** Easy

**Ссылка:** https://leetcode.com/problems/reverse-integer/

## Условие

Дано 32-битное знаковое целое число `x`. Нужно вернуть число, полученное разворотом цифр `x`. Если при развороте число выходит за пределы диапазона `[-2^31, 2^31 - 1]`, вернуть `0`.

## Примеры

| Вход  | Выход |
|-------|-------|
| 123   | 321   |
| -123  | -321  |
| 1200  | 21    |

## Ограничения

- `-2^31 <= x <= 2^31 - 1`

## Решение

### mathReverseInteger (shared)

Математическое разворачивание числа через деление и взятие остатка:

1. Сохраняем знак числа
2. Получаем модуль числа
3. В цикле: извлекаем последнюю цифру (`original % 10`) и добавляем к результату
4. Делим число на 10 (отбрасывая дробную часть)
5. Применяем знак к результату

### Проверка переполнения (isInt32)

После разворота проверяем `isInt32(reversed)`. Если результат выходит за пределы `[-2^31, 2^31-1]`, возвращаем `0`.

## Сложность

- Время: O(log₁₀(x)) — количество цифр в числе
- Память: O(1)

## Заметки

- `mathReverseInteger` — вынесена в shared, так как используется в нескольких местах
- `isInt32` — отдельный хелпер для проверки диапазона Int32
