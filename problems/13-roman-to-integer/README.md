# 13. Roman to Integer

**Сложность:** Easy

**Ссылка:** https://leetcode.com/problems/roman-to-integer/

## Условие

Римские цифры представлены семью различными символами: `I`, `V`, `X`, `L`, `C`, `D` и `M`.

| Символ | Значение |
|--------|----------|
| I      | 1        |
| V      | 5        |
| X      | 10       |
| L      | 50       |
| C      | 100      |
| D      | 500      |
| M      | 1000     |

Римские цифры обычно записываются от большего к меньшему слева направо. Однако цифра для четырёх — это не `IIII`, а `IV`. То есть единица перед пятёркой вычитается. Аналогично, для девяти используется вычитание: `IX` перед `X`. Вычитание используется в шести случаях:

- `I` перед `V` (5) и `X` (10) → `IV` (4) и `IX` (9)
- `X` перед `L` (50) и `C` (100) → `XL` (40) и `XC` (90)
- `C` перед `D` (500) и `M` (1000) → `CD` (400) и `CM` (900)

Дана римская цифра, преобразуйте её в целое число.

## Примеры

**Пример 1:**
```
Вход: s = "III"
Выход: 3
```

**Пример 2:**
```
Вход: s = "LVIII"
Выход: 58
Пояснение: L = 50, V = 5, III = 3
```

**Пример 3:**
```
Вход: s = "MCMXCIV"
Выход: 1994
Пояснение: M = 1000, CM = 900, XC = 90, IV = 4
```

## Ограничения

- `1 <= s.length <= 15`
- `s` содержит только символы: `I`, `V`, `X`, `L`, `C`, `D`, `M`
- Гарантируется, что `s` — валидная римская цифра в диапазоне `[1, 3999]`

## Решение

Алгоритм: идём по строке справа налево.

Для каждого символа сравниваем его значение со значением **следующего** символа (который находится правее):
- Если текущее значение **меньше** следующего — вычитаем
- Иначе — прибавляем

Такой подход автоматически обрабатывает вычитающие комбинации (`IV`, `IX`, `XL`, и т.д.).

## Сложность

- **Время:** O(n) — один проход по строке
- **Память:** O(1) — используется Map фиксированного размера и константное количество переменных

## Заметки

- Ключевая идея — движение справа налево позволяет за один проход определить, нужно ли вычитать или прибавлять
- Альтернативный подход: движение слева направо с проверкой следующего символа, но это требует дополнительной обработки последнего символа
