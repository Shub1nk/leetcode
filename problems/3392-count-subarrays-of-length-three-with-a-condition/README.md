# 3392. Count Subarrays of Length Three With a Condition

**Сложность:** Easy

**Ссылка:** https://leetcode.com/problems/count-subarrays-of-length-three-with-a-condition/

## Условие

Дан целочисленный массив `nums`. Вернуть количество **подмассивов длины 3** таких, что сумма первого и третьего элементов равна **ровно половине** второго.

Иначе говоря, для подмассива `[a, b, c]` (идущих подряд элементов) условие:

```
a + c == b / 2
```

## Примеры

**Пример 1**

```
Ввод:  nums = [1,2,1,4,1]
Вывод: 1
```

Подмассивы длины 3: `[1,2,1]`, `[2,1,4]`, `[1,4,1]`.
Подходит только `[1,4,1]`: `1 + 1 == 4 / 2` → `2 == 2`.

**Пример 2**

```
Ввод:  nums = [1,1,1]
Вывод: 0
```

Единственный подмассив `[1,1,1]`: `1 + 1 == 1 / 2` → `2 != 0.5`. Не подходит.

## Ограничения

- `3 <= nums.length <= 100`
- `-100 <= nums[i] <= 100`

## Решение

Один проход по всем тройкам подряд идущих элементов. Центр тройки `b` пробегает индексы `1 .. n-2`; для каждого проверяем условие.

Чтобы не работать с дробями, условие `a + c == b / 2` переписано в целых числах:

```
b === (a + c) * 2
```

Каждое выполнение условия увеличивает счётчик.

## Сложность

- Время: O(n) — один проход
- Память: O(1)

## Заметки

- Условие держим в целых числах (`b === (a + c) * 2`) вместо `a + c === b / 2` — без дробей и погрешностей.
