# is-int32

**Описание:** Хелпер для проверки, находится ли число в диапазоне 32-битного знакового целого.

**Диапазон Int32:** `[-2^31, 2^31 - 1]` = `[-2147483648, 2147483647]`

## Использование

```typescript
import { isInt32 } from "./shared/is-int32";

isInt32(42);      // true
isInt32(-100);    // true
isInt32(2147483647);  // true
isInt32(2147483648);  // false
```

## Сложность

- Время: O(1)
- Память: O(1)
