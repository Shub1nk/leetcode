# Разворот целого числа

**Тег:** math

## Описание

Функция для разворота целого числа: последовательное извлечение цифр справа налево и сборка их в новое число в обратном порядке.

## Сигнатура

```typescript
function mathReverseInteger(x: number): number
```

## Алгоритм

### Шаги

1. Сохранить знак числа
2. Взять модуль числа
3. Пока `original > 0`:
   - Добавить последнюю цифру к `reversed`
   - Сдвинуть `reversed` влево
   - Удалить последнюю цифру из `original`
4. Вернуть результат с сохранённым знаком

### Операции

- `x % 10` — получение последней цифры
- `Math.floor(x / 10)` — отбрасывание последней цифры

### Пример

```
Итерация | original | original % 10 | reversed
---------|----------|---------------|----------
старт    | 123      | -             | 0
1        | 12       | 3             | 0 * 10 + 3 = 3
2        | 1        | 2             | 3 * 10 + 2 = 32
3        | 0        | 1             | 32 * 10 + 1 = 321
```

### Формула

```
reversed = Σ(d_i × 10^i)  где i считается от 0
```

Для `123`: `(1 × 10²) + (2 × 10¹) + (3 × 10⁰) = 100 + 20 + 3 = 321`

### Почему это работает

Каждая цифра "выталкивается" из original в reversed:
- `original % 10` забирает последнюю цифру
- `reversed * 10` сдвигает уже накопленные цифры влево
- Складываем — новая цифра становится последней

## Примеры

```typescript
mathReverseInteger(123)   // 321
mathReverseInteger(-123)  // -321
mathReverseInteger(1200)  // 21
mathReverseInteger(0)     // 0
```

## Сложность

- **Время:** O(log₁₀(n))
- **Память:** O(1)

## Заметки

- Отрицательные числа сохраняют знак
- Ведущие нули теряются (1200 → 21)
- Переполнение не обрабатывается
